### Environment Development
* Netbeans 8.2
* OpenJDK 1.8
* Apache Maven 3.3.9 
* Fedora 25
### Running Project
1. Open a terminal
1. git clone https://github.com/heronsanches/animal-game.git
1. Go to "animal-game/animalGame" folder
1. mvn clean package
1. mvn exec:java -Dexec.mainClass="com.heronsanches.animalgame.AnimalGame"
